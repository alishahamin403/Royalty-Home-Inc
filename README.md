# Royalty Home Inc.

Static marketing site for Royalty Home Inc., focused on luxury home renovations in Toronto and the GTA.

## Stack

- Static HTML
- CSS
- Vanilla JavaScript

## Local Preview

```bash
python3 -m http.server 3000
```

Open `http://localhost:3000`.

## Public Repo Note

This repository is public for showcase purposes. Local machine files and private deployment details are intentionally excluded.
